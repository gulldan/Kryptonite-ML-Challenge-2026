# Исследовательские конфиги

Здесь лежат все конфиги, которые относятся только к исследовательской части проекта.

Основные разделы:

- `training/` — активные training и fine-tuning профили;
- `release/` — speed, TensorRT и release-like исследовательские прогоны;
- `deployment/` — внутренние smoke-конфиги для runtime boundary.

Финальный конфиг для организаторов сюда не кладётся. Его место — корневой `configs/submission.toml`.
