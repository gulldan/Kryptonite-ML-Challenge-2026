# Исследовательские документы

Папка содержит всю исследовательскую документацию проекта.

Куда смотреть:

- `challenge-experiment-history.md` — компактная таблица результатов на лидерборде;
- `trails/` — подробные записи запусков, гипотез и диагностик;
- `challenge-solution-report.md` — архивный итоговый отчёт;
- `release-runbook.md` — архивный runbook для ветки MS41;
- `inference-acceleration.md` — исследования скорости и TensorRT;
- `data.md`, `training.md`, `configuration.md`, `model-card.md` — рабочие заметки;
- `assets/` и `reference/` — исследовательские графики и справочные материалы.

Что не находится здесь:

- финальная organizer-facing документация;
- инструкции финального запуска;
- описание корневого `Dockerfile`.

Для этого предназначен корневой каталог `docs/`.
