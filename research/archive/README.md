# Архив

Здесь лежат материалы, которые важно сохранить, но которые не должны мешать чтению
корня репозитория как organizer-facing поставки.

Сюда входят:

- `organizer-baseline/` — архивный baseline организаторов;
- `evidence-bundles/` — зафиксированные bundles старых кандидатов;
- `org-materials/` — исходные PDF и другие материалы от организаторов.
