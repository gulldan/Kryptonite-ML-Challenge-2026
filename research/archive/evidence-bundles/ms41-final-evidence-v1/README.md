# Финальный пакет доказательных материалов MS41

Документарный релизный пакет для финального кандидата
`MS41_ms32_classaware_c4_weak_20260415T0530Z`.

Этот комплект нужен для передачи итоговых материалов, когда в текущем локальном
рабочем каталоге нет полного набора рантайм-артефактов. В него входят письменные
материалы по релизу, релизные конфиги, ближайшие локально доступные рантайм-записи
родительской ветки (`MS32`), а также машиночитаемая запись недостающих путей,
SHA и временных меток для `MS41`.

Состав:

- `submission_bundle.md` -- человекочитаемое резюме
- `submission_bundle.json` -- машиночитаемый манифест
- `sources/runtime_external_references.json` -- зафиксированные финальные пути рантайма для `MS41`
