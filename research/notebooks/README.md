# Ноутбуки

Папка предназначена для исследовательских ноутбуков.

## Что здесь есть

- [eda.ipynb](./eda.ipynb) — финальный EDA notebook для анализа данных и артефактов.

Правила для этой папки:

- production-логика не должна жить только в ноутбуке;
- переиспользуемый код нужно выносить в `src/kryptonite/` или `research/scripts/`;
- ноутбук может использоваться как удобная точка входа в EDA, но не как источник
  истины для pipeline.
