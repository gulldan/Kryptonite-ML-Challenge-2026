# Исследовательские скрипты

Папка предназначена для исследовательских CLI-скриптов.

Сюда относятся:

- training и fine-tuning entrypoints;
- EDA и диагностические команды;
- benchmark, export и speed-утилиты;
- вспомогательные скрипты подготовки манифестов и внешних данных.

Финальные команды для организаторов должны лежать в корневом `scripts/`.
