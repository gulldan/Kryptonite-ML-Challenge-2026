# Скрипты

Папка содержит только финальные CLI-команды organizer-facing пути.

Сейчас здесь лежит:

- `validate_submission.py` — канонический валидатор формата `submission.csv`.
- будущий `run_submission.py` — основной runner финального решения, который появится после выбора модели.

Что не должно лежать здесь:

- training, EDA, benchmark, export и speed-скрипты;
- диагностические и smoke-команды;
- внутренние утилиты для экспериментов.

Всё это вынесено в `research/scripts/`.
